# loofabag

A Next.js landing page for the myloofabag brand, migrated from the original static HTML homepage.

## Getting started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Vercel compatibility

This project is ready for Vercel deployment using the Next.js standard build process.

- `npm run build` generates the production build
- Vercel automatically detects the Next.js project

## Pages

- `app/page.tsx` contains the migrated landing page content
- `app/globals.css` contains the styles ported from the original site
